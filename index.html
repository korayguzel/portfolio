About Me

I graduated in Molecular Biology and Genetics from Istanbul Technical University. During my master's program at Acıbadem University (Molecular and Translational Biomedicine), I worked on aging research with long-lived cell models.

In my second year, I joined Rizzoli Lab at University Medicine Göttingen as a guest researcher — analyzing proteomic and transcriptomic data on brain aging, performing immunofluorescence staining and cryostat sectioning in the wet lab, and building R/Bioconductor pipelines (DESeq2, WGCNA) on the computational side.

After returning, I left the master's program to pursue independent research. Now I build open-source computational tools for biological problems. My approach: systematically survey the literature, find gaps in existing tools, learn the necessary techniques, and ship a working tool.

I'm not a proficient programmer — I vibe code with Claude Code and know enough to steer and validate the output.

Physics-based quality control for AI-designed proteins

MD simulation (OpenMM) + cavity detection (fpocket) + H-bond persistence + secondary structure stability + SASA check + composite PASS/WARNING/FAIL verdict. Interactive CLI, AI chat assistant with 8 LLM providers, FastQC-style HTML reports, CUDA GPU accelerated, Docker support, 250 tests.

Vision-Language Model for cellular senescence detection

Detects cellular senescence from fluorescence microscopy (IF) images using a Vision-Language Model — the first VLM applied to this domain in the literature. QLoRA fine-tuning with SenNet consortium data. Qwen3-VL base model, Phi-4-reasoning-vision for annotation distillation.
